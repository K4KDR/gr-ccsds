Detailed Description

Takes I frames and generates the transmitted codeblock out of it.

The block has an input message port named "in" at which frames of length 223*I bytes are received. On reception the frames are interleaved and encoded. The generated codeblock consisting of 255*I bytes is then queued in the output message port named "out".
