See Also
d_POLY_C1
d_POLY_C2

The generator polynomial is a condensed information about how to combine the shift register outputs and whether or not to invert the output stream.

If the MSB is set the output stream will be inverted.

The lower 7 bits encode the actual polynomials. The generator polynomial is masked with 0x7F (set MSB to zero) and the copied into the d_poly_x variables.

uint16_t ccsds_conv_encode27_punct_aux::d_state
private

State information vector.

Counting from 0, starting at the LSB this state contains: bit0 - bit6 : Next input byte (that is not processed yet) bit7 : Current information bit i(t) bit8 : i(t-1) bit9 : i(t-2) bit10 : i(t-3) bit11 : i(t-4) bit12 : i(t-5) bit13 : i(t-6) bit14 : i(t-7) bit15 : Allways zero to esnure no overfloat when shifting the state left.
