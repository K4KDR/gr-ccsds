CCSDS_API ccsds_dll_cc_sptr ccsds_make_dll_cc ( unsigned int  osf,
float  gamma 
)

Return a shared_ptr to a new instance of ccsds_dll_cc.

Create an instance of ccsds_dll_cc and return it's shared_ptr.

Parameters
osfSignal oversampling factor. A factor of two means that two samples contain one symbol. The oversampling factor also defines the decimation factor of this block.
gammaBlock's filter stepsize. Each estimated timing error will be multiplied by gamma and then added to the Delay estimate. A high value (<=1) will allow the loop to follow timing changes very quickly (instantly for 1), but be very sensitive to noise, while lower values will reduce the noise impact but will take longer to settle for a certain delay
Returns
Shared pointer to the created DLL block
