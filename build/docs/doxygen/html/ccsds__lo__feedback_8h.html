Detailed Description

Defines all parameters for the frequency feedback loop.

Defines the parameters to be used by the frequency correction messages and the frequency offset tags used by ccsds_local_oscillator_cc.

See Local oscillator frequency correction feedback loop for more details.

#define MSG_FREQ_ARG2   -1.42

Value used for message's arg2 when sending frequency corrections to the local oscillator.

Arbitrary chosen number to be used for sending frequency correction values to the local oscillator. This value is checked by the local oscillator and the message is discarded if arg2 does not match. (Sanity check)
