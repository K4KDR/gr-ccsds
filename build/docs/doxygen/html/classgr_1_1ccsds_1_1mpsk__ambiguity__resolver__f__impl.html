Public constructor of the AR.

Parameters
MModulation order. Determins how many parallel streams to watch.
ASMString containing the attached sync marker to look for in hexadecimal representation. Must be of even length.
asm_lenLength of the ASM in bits
thresholdNumber of ASM losses to enter full search again. If set to one, a full search is performed every time an ASM is lost.
correlation_thresholdMinimum normalized correlation between an incomming bit sequence and the ASM to consider them matching.
num_tail_symsNumber of bits after the Frame that should be copied as well. Default is zero, so only the frame data is copied.
frame_lengthLength of a frame (without ASM) in bytes.

Constructs a AR block that searches for the ASM in the M input streams and outputs the one that contains the ASM. If no ASM is found, no output is given. This block preserves the ASM, as it is still needed for frame synchronization after decoding.

Reads all tags from the buffer and creates a frame metadata header out of it.

Calculates the reception time of the sample with the absolute count from based on the tag rx_time. Otherwise ignore the rx_time tag and label the calculated reception time rx_time in the created metadata.

Copies all other tags into the header.

Parameters

lenLength of the frame. Consider all tags up to from + len.
Returns

const unsigned int  num_in,

)
private

Search for ASM in input sequence and return maximum found correlation.

The search checks all possible offsets in the range of [0, num_in-d_ASM_LEN_BITS] starting from zero and abborts if a correlation of 1.0f is reached, or all offsets have been tried.

Parameters
inArray of num_in input softbits. Must contain at least offset+d_ASM_LEN_BITS elements.
num_inNumber of elements in in. Must be at lease d_ASM_LEN_BITS
offsetPointer to an unsigned int where the offset of the found maximum correlation is stored.
Returns
The maximum found correlation.
