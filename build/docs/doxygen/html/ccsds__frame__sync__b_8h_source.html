64 /*!
65  * \brief Take the input streams and look for an ASM.
66  *
67  * \ingroup receiver
68  *
69  * The block will start in searchmode, looking for every possible bit and byte
70  * offset. Once an ASM is found it will enter locked state where it will only
71  * check for the ASM at the expected position. If the ASM is found a counter is
72  * increased up to \c d_THRESHOLD. If an exprected ASM is not found the counter
73  * is decreased. If the counter reaches zero the block goes into search mode
74  * again.
75  *
76  * If the block finds an ASM it will create an asynchronous message containing
77  * the \c frame_length bytes after the ASM. The input stream is copied the
78  * output stream in any case.
79  */
