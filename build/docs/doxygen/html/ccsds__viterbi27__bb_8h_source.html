38 /*! \brief A rate 1/2, k=7 convolutional frame decoder for the CCSDS standard
39  * \ingroup channelcoding
40  *
41  * This block performs hard-decision convolutional decoding using the Viterbi
42  * algorithm. The fec library by Phil Karn is used.
43  *
44  * The input is a stream of unpacked bits (hard channel symbols) that represent the encoded frames only,
45  * i.e. the ASMs have been removed from the stream, and are synchronised on the first bit of the frame.
46  *
47  * There is no stream output, instead the MSB-first packed bytes of the decoded
48  * frame are inserted into a dedicated output queue.
49  *
50  * This block is designed for framed data, not continuous data streaming.
51  */
