CCSDS_API ccsds_mpsk_demod2_cb_sptr ccsds_make_mpsk_demod2_cb ( const unsigned int  M,
const unsigned char  quant_bits = 0 
)

Create an M-PSK demodulator and return it's shared pointer.

Parameters
MModulation order of the M-PSK signal.
quant_bitsConfidence value quantization in bit of the detected symbols. Must be in the interval [0,8].
Returns
Shared pointer of the created detector.

If quant_bits is zero, only the detected symbols are generated. If set to nonzero a second output stream at the same rate as the detected symbols will be generated containing a quant_bits bit value representing the confidence in this detection, where 0 means this symbol is a guess as good as any other symbol and (2 ^quant_bits) - 1 means highest confidence in the detected symbol.

The confidence value is generated by a linear quantization of the symbol's squared magnitude in the range from [0.0 to 1.0]. Any symbol with a squared magnitude higher than 1.0 will have highest confidence as well.
