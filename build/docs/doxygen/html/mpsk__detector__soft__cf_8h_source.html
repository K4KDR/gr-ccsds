11  /*! \brief M-PSK soft bit detector.

13  *
14  * M-PSK soft bit detector.
15  *
16  * The incomming phase is converted to a conteallation point (counting
17  * counterclockwise starting from zero at symbol 1+0j) with the non linear
18  * function:
19  *
20  * \code const_point = floor( 0.5*M*fmod( angle/pi + 2, 2) + 0.5 ) \endcode
21  *
22  * And then the constellation point is mapped to the actual symbol bit sequence
23  * by a binary to gray code mapping [http://en.wikipedia.org/wiki/Grey_code#Converting_to_and_from_Gray_code]:
24  *
25  * \code data = (const_point >> 1) ^ const_point \endcode
26  *
27  * For each incomming symbol \c ld(M) output symbols are generated containing
28  * the detected soft bit as float values, where -1.0f represents the most
29  * confident 0 bit, 0.0f is an erasure (in this case, the received symbol has
30  * no energy at all) and 1.0f represents the most confident 1 bit.
31  */
