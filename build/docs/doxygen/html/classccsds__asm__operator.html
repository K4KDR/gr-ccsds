Copy bitstream from a byte array with a bit offset into an aligned byte array.

Parameters
stream_inArray of bytes to copy. First bit to copy is located in the first element of this array. If offset_bit is zero this array must contain at least len elements, if offset_bit is greater thatn zero, this array must contain at least len+1 elements.
stream_outArray in which the resulting bytes should be copied. Memory for at least len elements must be allocated.
lenNumber of bytes to copy.
offset_bitsNumber of bits to skip in the first byte, starting with the MSB. If offset_bit is zero this function copies the first len bytes from stream_in to stream_out by using memcpy. Otherwise the copy operation is done manually using bitwise operations.

The first offset_bits starting with the MSB from the first element of stream_in are skipped, afterwards all bits from stream_in are copied to stream_out until len bytes have been copied to stream_out.

bool ccsds_asm_operator::search_asm ( const unsigned char *  stream,
const unsigned int  stream_len,
unsigned int *  offset_bytes,
unsigned int *  offset_bits 
)
inline

Searches for ASM in the given stream.

Parameters
streamArray of bytes in which to search for the ASM.
stream_lenLength of the stream. The last d_ASM_SYM_LEN bytes of this stream are not searched (as the ASM would only fit in partially).
*offset_bytesPointer to an unsigned int where the offset bytes between start of stream and the found ASM should be stored, if a match is found.
*offset_bitsPointer to an unsigned int where the offset bits between start of stream and the found ASM should be stored, if a match is found.
Returns
true if ASM is found, false otherwise.

The match with the minimum Hamming distance is searched. If this minimum Hamming distance is less or equal the d_BER_THRESHOLD a match is found, otherwise no match is found and d_offset_bytes and d_offset_bits are set to zero.

In case there are two matches with the same Hamming distance the one with the lower byte offset or lower bit offset (in this priority) is chosen.
