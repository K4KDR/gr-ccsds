gr::ccsds::dll_cc_impl::dll_cc_impl ( unsigned int  osf,
float  gamma 
)

Public constructor of the DLL.

Parameters
osfSignal oversampling factor. A factor of two means that two samples contain one symbol. The oversampling factor also defines the decimation factor of this block.
gammaBlock's filter stepsize. Each estimated timing error will be multiplied by gamma and then added to the Delay estimate. A high value (<=1) will allow the loop to follow timing changes very quickly (instantly for 1), but be very sensitive to noise, while lower values will reduce the noise impact but will take longer to settle for a certain delay Constructs a DLL block with the given oversampling factor and loop stepsize.

double gr::ccsds::dll_cc_impl::d_mu
private

Storage for the current fractional index .

The fractional index is in the range [0,1) and shows where in between the basepoint sample and the following sample the interpolation shall be done. A fractional index of 0 means the interpolant is equal to the basepoint sample while 0.5 means the interpolated sample lies exactly in the middle between the basepoint sample and the following sample.
