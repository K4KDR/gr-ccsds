34 /*!
35  * \brief Given 2 parallel streams of bits and access_code flags, assemble TM Frames.
36  * \ingroup receiver
37  * \ingroup synchronization
38  *
39  * It is a fact of non-differential BPSK demodulation that an inverted bit stream may be produced.
40  * Two parallel gr_correlate_access_code_bb blocks have been setup, one looking for the non-inverted ASM,
41  * and the other for the inverted ASM. If the framer sink detects a frame marker on the inverted stream,
42  * it inverts the input.
43  *
44  * There is no stream output, instead the MSB-first packed bytes of the synchronised
45  * frame are inserted into a dedicated output queue.
46  *
47  * The input data consists of bytes that have two bits used.
48  * Bit 0, the LSB, contains the data bit.
49  * Bit 1 if set, indicates that the corresponding bit is the
50  * the first bit of the packet. That is, this bit is the first
51  * one after the access code.
52  */
