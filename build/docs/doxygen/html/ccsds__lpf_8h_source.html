8 /*! \brief Create a linear first order low pass filter and return it's shared
9  * pointer.
10  *
11  * \param loop_bw Bandwidth of the loop, which determins the loop stepsize.
12  * \return Shared pointer to the created filter.
13  *
14  * Based on the loop bandwidth the filter stepsize alpha is calculated
15  * with: [\ref bib_mengali p. 126 (3.5.67)]
16  * \code alpha = 4 * loop_bw \endcode
17  *
18  * The k-th filtered state x from the inputs y is calculated as
19  * \code x[k] = (1-alpha) * x[k-1] + alpha * y[k] \endcode
20  */
21 lpf *ccsds_make_lpf(float loop_bw);
