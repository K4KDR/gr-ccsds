13 /*! \brief Create a Convolutional encoder block and return it's shared pointer.
14  *
15  * \param gen_poly_c1 Compact generator polynom for c1 stream. Default is 0x79.
16  * \param gen_poly_c2 Compact generator polynom for c2 stream. Default is 0xDB.
17  * \param puncturing_type Symbol for type of puncturing. Default is NONE.
18  * \return Shared pointer of the Convolutional encoder block.
19  *
20  * The compact generator polynom contains two informations. Encoded in the MSB
21  * is the information whether the stream should be inverted (MSB set) or not
22  * (MSB unset). Encoded in the 7 less significant bits is the information about
23  * the connections between the shift registers and the output. The MSB-1
24  * represents the influence of the current information bit i(t) while the LSB
25  * represents the influence of the information bit i(t-6). A set bit means
26  * there is a connection, while an unset bit means there is no.
27  *
28  * The polynomials for the convolutional code defined in ECSS can be generated
29  * with 0x79 for c1 and 0x5B for c2. Together with the MSB for the inversion of
30  * c2 the compact polynomials are 0x79 and 0xDB.
31  */
