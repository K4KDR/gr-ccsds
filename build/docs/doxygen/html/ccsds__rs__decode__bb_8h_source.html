39 /*! \brief A (255,223) Reed-Solomon decoder for CCSDS TM Frames
40  * \ingroup channelcoding
41  *
42  * The forward error correction library by Phil Karn is used.
43  *
44  * The input is a stream of R-S symbols (8 bit packed bytes) that represent the codeblocks (encoded frames) only,
45  * i.e. the ASMs have been removed from the stream, and are synchronised on the first symbol of the block.
46  *
47  * There is no stream output, instead the packed bytes of the decoded
48  * frame are inserted into a dedicated output queue (minus the R-S parity bits).
49  * Frames with uncorrectable errors are not propagated.
50  */
