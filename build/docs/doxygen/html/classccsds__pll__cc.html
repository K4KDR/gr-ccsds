Detailed Description

Phase locked loop that sends frequency correction messages.

See Also
Local oscillator frequency correction feedback loop
ccsds_lo_feedback.h

Phase locked loop using the Viterbi&Viterbi method to estimate the sample's phase offset. Estimates are filtered with a linear second order low pass filter.

After reception of a frequency_offset tag the loop waits for another PLL_FREQ_UPDATE samples before a new frequency correction message is send out to close the frequency feedback loop.

Frequency corrections are send out on the port "freq".

Local oscillator frequency correction feedback loop
ccsds_lo_feedback.h

If ccsds_pll_cc::d_lo_msg_tag is true the function will return immediatley. This means that the loop has allready eceived a frequency_offset tag within the last PLL_FREQ_UPDATE samples and is now waiting for the counter to hit zero so a new frequency correction message can be send.

If ccsds_pll_cc::d_lo_msg_tag is false, the funktion will look for frequency_offset tags in range [from,from+num) and if any is found the earliest's relative offset to from plus PLL_FREQ_UPDATE is stored in ccsds_pll_cc::d_lo_msg_offset and ccsds_pll_cc::d_lo_msg_tag is set to true. This will cause the loop to process another PLL_FREQ_UPDATE samples before new frequency correction messages are send.
