4 /*! \brief Moving average filter
5  * \ingroup filter
6  *
7  * A dual moving average filter. The filter calculates the moving average of an
8  * input sequence. In order to not accumulate numerical computation errors the
9  * filter is reset after processing moving_average::D_NUM_RESET samples. A
10  * second (inactive) moving average filter starts to filter the input stream in
11  * parallel moving_average::D_NUM_COLLECT samples before the first (active)
12  * filter is reset.
13  *
14  * The filtered output is always taken from the active filter.
15  *
16  * When the active filter is reset, the inactive filter has filtered exactly
17  * moving_average::D_NUM_COLLECT samples and becomes active while the
18  * previously active filter becomes inactive.
19  */
