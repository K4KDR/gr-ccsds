CCSDS_API ccsds_mpsk_ambiguity_resolver_f_sptr ccsds_make_mpsk_ambiguity_resolver_f ( const unsigned int  M,
std::string  ASM,
const unsigned int  asm_len,
const unsigned int  threshold,
const float  correlation_threshold,
const unsigned int  frame_length,
const unsigned int  num_tail_syms = 0 
)

Return a shared_ptr to a new instance of ccsds_mpsk_ambiguity_resolver_f.

Create an instance of ccsds_mpsk_ambiguity_resolver_f and return it's shared_ptr.

Parameters
MModulation order. Determins how many parallel streams to watch.
ASMString containing the attached sync marker to look for in hexadecimal representation. If asm_len is not a multiple of 8 (i.e. ASM is not a multiple of bytes), the ASM string should be tailed by zeros.
asm_lenLength of the ASM in bits
thresholdNumber of ASM losses to enter full search again. If set to one, a full search is performed every time an ASM is lost.
correlation_thresholdMinimum normalized correlation between an incomming bit sequence and the ASM to consider them matching.
frame_lengthLength of a frame (without ASM) in bytes.
num_tail_symsNumber of bits after the Frame that should be copied as well. Default is zero, so only the frame data is copied.
Returns
Shared pointer to the created AR block
