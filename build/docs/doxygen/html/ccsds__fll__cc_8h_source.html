143  /*! \brief Calculate the samplewise phase difference as a rotator.
144  *
145  * \param tmp_c Aligned complex array in which the differences should
146  * be stored. Memory for at least \c num elements must be allocated.
147  * \param in Complex array of samples which's differences should be
148  * computed. Must contain at least \c num elements.
149  * \param num Number of samples to process.
150  *
151  * Implements the delay and multiply frequency error detector
152  * tmp_c[k] = in[k] * conj(in[k-1])
153  * where conj( . ) computes the complex conjugate.
154  *
155  * The algorithm assumes a complex sample 1+0j at the very beginning
156  * and stores the last sample per block for the next iteration after
157  * the first run. This way \c num input samples generate \c num output
158  * samples without going out of array bounds.
159  *
160  * This block is using volk, \c tmp_c \b must be aligned, \c in \b
161  * should be aligned for better performance.
162  */
163  void calc_diffs(gr_complex *tmp_c, const gr_complex *in, const unsigned int num);
