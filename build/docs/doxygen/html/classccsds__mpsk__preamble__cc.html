Detailed Description

Append a M-PSK preamble before the input stream.

Parameters
numNumber of preamble symbols.

Outputs num preamble symbols, before the input stream is copied to the output stream. This can be used to let the receiver synchronize before sending frames. This way even the first transmitted frame can be received and decoded.

The preable consists of alternating BPSK symbols (starting with 1+0j). BPSK symbols are also part of any other M-PSK constellation where M is a power of two and thus works for all M-PSK receivers.
