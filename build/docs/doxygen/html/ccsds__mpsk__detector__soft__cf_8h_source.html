19 /*! \brief M-PSK soft bit detector.
20  * \ingroup receiver
21  *
22  * M-PSK soft bit detector.
23  *
24  * The incomming phase is converted to a conteallation point (counting
25  * counterclockwise starting from zero at symbol 1+0j) with the non linear
26  * function:
27  *
28  * \code const_point = floor( 0.5*M*fmod( angle/pi + 2, 2) + 0.5 ) \endcode
29  *
30  * And then the constellation point is mapped to the actual symbol bit sequence
31  * by a binary to gray code mapping [http://en.wikipedia.org/wiki/Grey_code#Converting_to_and_from_Gray_code]:
32  *
33  * \code data = (const_point >> 1) ^ const_point \endcode
34  *
35  * For each incomming symbol \c ld(M) output symbols are generated containing
36  * the detected soft bit as float values, where -1.0f represents the most
37  * confident 0 bit, 0.0f is an erasure (in this case, the received symbol has
38  * no energy at all) and 1.0f represents the most confident 1 bit.
39  */
