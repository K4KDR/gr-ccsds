lpf2* ccsds_make_lpf2 ( const double  loop_bw,
const double  damping_factor_squared,
const double  s_slope 
)

Create a linear second order low pass filter and return it's shared pointer.

Parameters
loop_bwBandwidth of the loop. Used to compute the loop stepsizes.
damping_factor_squaredSquared value of the loop's damping factor. Used to compute the loop stepsizes.
s_slopeEstimated slope of the S-curve of the error detector at the origin. Used to compute the loop stepsizes.
Returns
Shared pointer of the created filter.

The loop stepsizes gamma and rho are given by the following formulas: [Mengali & D'Andrea p. 220 ff.]

loop_bw = ( 2*rho + gamma*s_slope*(2+rho) ) / ( 8 - 2*gamma*s_slope*(2+rho) ) (5.3.92)
damping_factor = ( (1+rho)*sqrt(gamma*s_slope) ) / ( 2*sqrt(rho)) (5.3.86)

The stepsize rho can not be computed in explicit form and is computed with Newton's method by solving the root of the following polynomial:

f(rho) = rho^3 + a * rho^2 + b * rho + c != 0

with

a = - 4*loop_bw + 4*loop_bw*damping_factor_squared + 2 + 2*damping_factor_squared
b = - 8*loop_bw + 8*loop_bw*damping_factor_squared + 1 + 4*damping_factor_squared
c = - 4*loop_bw

Given rho, gamma is computed in closed form as

gamma = ( 4*damping_factor_squared*rho ) / ( s_slope * (1+rho)^2 )
